# Example Outputs

These are **illustrative examples** showing what each skill produces when run against a realistic product scenario. All examples use the same fictional company — **Velora** — to demonstrate how skills chain together.

> **About Velora (fictional):** A Series A developer tool startup building an AI code review platform. 25 employees, $8M ARR, 2,000 active teams. Competes with GitHub Copilot code review, CodeRabbit, and Sourcery. Target market: mid-size engineering teams (20-100 devs) shipping 50+ PRs/week.

| Example | Skill Used | What It Demonstrates |
|---|---|---|
| [Customer Discovery Report](customer-discovery-example.md) | `customer-discovery` | Interview guide, hypothesis tracker, synthesis framework, discovery report with validated/invalidated hypotheses |
| [Competitive Response Playbook](competitive-response-example.md) | `competitive-response` | Threat assessment, competitive matrix, response playbook, moat analysis |
| [Feature Prioritization Memo](feature-prioritization-example.md) | `feature-prioritization` | Scored feature matrix, trade-off documentation, cost-of-delay analysis, sequenced roadmap |

These examples were generated by running the actual skills. Your outputs will be grounded in **your** company's context — these show the structure and depth you can expect.
