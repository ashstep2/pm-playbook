# Founders Context

One file per founder/key leader. Include:
- Attributed quotes from public talks, podcasts, blog posts
- Key themes and recurring messages
- Background and expertise areas
- Source URLs for every quote

Format:
```markdown
# [Name] — [Title]

## Key Themes
- Theme 1: description
- Theme 2: description

## Attributed Quotes

### [Topic]
> "Direct quote"
> — *Context* ([source](url))
```
